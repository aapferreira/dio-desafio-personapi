# Desenvolvendo um sistema de gerenciamento de pessoas em API REST com Spring 

### Julho/2022
DIO - Resolução do desafio
- Seguindo as instruções do professor, copiando, colando, testando e vendo resultados.
- Java 11, Spring Boot DevTools, Lombok, Actuator, Validation, Spring Data JPA, Spring Web, H2, Heroku, MapStruct
- Camadas: Controller, Service, Model, Repository, DTO, Mapper


Para executar o projeto no terminal, digite o seguinte comando:

```shell script
mvn spring-boot:run 
```

Após executar o comando acima, basta apenas abrir o seguinte endereço e visualizar a execução do projeto:

```
http://localhost:8080/api/v1/people
```
